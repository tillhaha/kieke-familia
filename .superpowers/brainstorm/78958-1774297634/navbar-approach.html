B — Bottom Navigation Bar

Nav moves to the bottom of the screen on mobile, like a native app. Top bar keeps only logo + user.

Pros

  • Thumb-friendly
  • Always visible, no extra tap

Cons

  • More complex to implement
  • Less familiar on web apps

C — Compact Inline Nav

Abbreviate nav labels on small screens, hide the user dropdown behind an icon. No menu toggle needed.

Pros

  • Simplest to implement
  • Always one tap away

Cons

  • Tight on very small phones
  • May not fit all link labels
